Number

Simple input component to handle numbers.

Samples

{{ number | json }}

Options

Name Type Default Description
ngModel string Component's model.
disabled boolean false Whether the input is disabled.
placeholder string Placeholder number.
decimals number Number of decimal numbers. Leave blank if you want to return all decimals available.
decimalSeparator string "," Decimal separator sign.
thousandSeparator string "." Thousand separator sign.
