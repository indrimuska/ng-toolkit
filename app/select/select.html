Options

Name Type Default Description
ngModel any | any[] Component's model.
disabled boolean false Whether the input is disabled.
multiple boolean false When set to true, it allows to select more than one option (the model will be an array of the selected values).
placeholder string Placeholder text.
options any[] Options available in the input.
labelAttr string Property name used to show an option. This should be a property available in every item of the options array. Leave blank if you want to show the entire option (if printable).
valueAttr string Property name used for the value of the option. This should be a property available in every item of the options array. Leave blank if you want to get the entire option.
